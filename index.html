Quadro compartilhado

—

Todos os aparelhos que usarem este código veem o mesmo quadro. Quem editar, atualiza o dos outros.

É o endereço do Realtime Database do seu projeto. Sem ele, o quadro fica só neste aparelho.

Como funciona. O quadro inteiro é gravado como um registro só, sob o código escolhido. A cada 8 segundos o app confere se outro aparelho mudou alguma coisa e traz a versão mais recente. Se dois aparelhos editarem ao mesmo tempo, vale a edição mais recente — combinem quem mexe em quê.
